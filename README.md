## Overview  
Planning a trip or daily activities becomes easier with real-time weather updates! **Weather Dashboard - Get Ready** provides **current** and **5-day weather forecasts** for any city worldwide. This application fetches weather data using **server-side APIs**, helping users make informed decisions before heading out.  

## Features  
✅ Search for any city worldwide 🌍  
✅ View **current weather** (temperature, humidity, wind speed, conditions) ☀️  
✅ See a **5-day forecast** with weather icons 📅  
✅ Track **search history** and revisit past searches 📌  
✅ Responsive design for desktop & mobile users 📱  

## Technologies Used  
- **Front-end:** HTML, CSS, JavaScript  
- **APIs:** OpenWeather API for real-time weather data  

## Installation & Usage  
For local:
### 1️⃣ Clone the Repository  
```sh
git clone git@github.com:san1718/mc06_WeatherDashboard.git
cd mc06_WeatherDashboard
```

### 2️⃣ Open the Application  
Simply open `index.html` in your preferred browser.  

OR, start a local server:  
```sh
npx live-server
```

## Summary  
The **Weather Dashboard** provides a simple and efficient way to check the **current weather** and **5-day forecast** of any city. By leveraging real-time weather APIs, users can plan their travels or daily schedules accordingly.
